## 在 Zeabur 上部署

推荐使用 Zeabur，但是需要执行几个步骤才能完成部署

1. Fork 这个仓库为你自己的仓库
2. 在 [Zeabur](https://docs.zeabur.com/zh-CN/get-started) 部署你的仓库
